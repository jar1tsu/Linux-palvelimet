# Oma Linux

## a) Raportin kirjoittaminen
Raportointi pitää sisällään kokonaisuudessaan sen mitä teit tarkasti ja mitä tapahtui. Ne pitää olla kerrottu. Raporttia kannattaa kirjoittaa samaan aikaan kun teet tehtävää/projektia. Raporttia voi kirjoittaa monesa muodossa, mutta nykypäivänä on helpointa tehdä se sähköiseen muotoon. (Karvinen 2006.)

## Toistettavuus:
- Raportin tulee olla tarkka, niin että, toinen opiskelija toistaa samat tulokset
- Myös virheet kannattaa raportoida
- Kerro myös ympäristösi (paikka, aika, laitteisto, käyttöjärjestelmä jne.)

## Täsmällisyys:
- Kirjaa tarkat komennot ja mitä teit
- Merkitse ajankohta
- Kerro missä onnistui, missä ei. Myös jatkotoiminnat, eli mitä jatkossa tapahtuu. Kirjoita kaikki mahdollisimman tarkasti.
- Raportoi myös odottamattomat tulokset → voivat paljastaa ohjelmistovirheitä
- Kirjoita imperfektissä: “suoritin”, “valitsin”, “ajoin komennon”

## Helppolukuisuus:
- Käytä väliotsikoita
- Kirjoita selkeällä kielellä
- Lisää tiivistelmä ja loppu kappale, jos on tarve

## Viittaukset:
- Kerro aina mistä tieto tulee
- Ilmoita lähteet selkeästi
- Raporttiin voi lisätä lisenssitekstit

## Mokia
- Sepittäminen: ei saa keksiä testejä tai tuloksia → se on vilppiä.
- Plagiointi: ei saa kopioida tekstiä tai kuvia ilman lähdettä.
- Kotitehtävien tulee olla pääosin opiskelijan omaa työtä.
- Luvaton kopiointi rikkoo tekijänoikeuksia.

## Debian Linux

Suoritin tehtävän 18.1.2026. Omalla koneella MacBook pro m2 2023, jossa on MacOS Tahoe 26.2. Pyöritän Debiania VirtualBoxissa, jonka versio on 7.2.4. Latasin netistä uusimman version, minkä sai minun koneelle, joka on siis ARM 64-bit. 

Latasin netistä Debian ISO-kuvan(ARM64). Lisäsin ISO-kuvan uuteen machineen. Nimeksi tuli Debian, OS on debian ARM64. Muistia (RAM) 4096 MB. Virtuaalilevyn koko on 20GB. Valitsin Graphical Install ja määritin: 
- Kieliasetukset
- Näppäimistö
- Verkkoasetukset
- Käyttäjänimi ja salasana

<img width="2314" height="1576" alt="image" src="https://github.com/user-attachments/assets/86b21f02-a904-468e-b6ce-3115b756eb4c" />

Asennus eteni ilman suurempia ongelmia ja tiedot virtuaalikoneen määirtykseen sain sivulta: https://terokarvinen.com/2021/install-debian-on-virtualbox/

Kirjauduin sisään luomilla tunnuksillani ilman ongelmia. Tämän jälkeen avasin terminaalin ja suoritin komennon "sudo apt-get update". Se hakee internetistä tuoreimman listan saatavilla olevista ohjelmista ja niiden versioista. Sen jälkeen suoritn komennon "sudo apt-get -y dist-upgrade", joka asentaa kaikki saatavilla olevat päivitykset. 

Näiden jälkeen käytin terminaalissa komentoa "sudo apt-get -y install ufw", joka asentaa palomuuriohjelman järjestelmään. Sen jälkeen aktivoin sen: sudo ufw enable ja käynistin koneen, jotta kaikki toimii.
