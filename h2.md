# Command Line Basics Revisited

- Linuxin komentorivi on vanha, nopea, tehokas

## Tärkeimmät peruskomennot:
#### - pwd, ls, cd: liikkuminen hakemistoissa
#### - less: tiedostojen katselu
#### - nano: helppokäyttöinen tekstieditori
#### - mkdir, mv, cp, rm: hakemistojen ja tiedostojen käsittely

- ssh ja scp mahdollistavat turvallisen etäyhteyden ja tiedostojen siirron
- --help tarjoavat komentokohtaisen ohjeistuksen
- Tab‑täydennys ja komentohistoria nopeuttavat työskentelyä
- Tärkeimmät hakemistot: /home, /etc, /var/log, /media
- sudo tarvitaan järjestelmätason tehtäviin
- Ohjelmien asennus hoidetaan paketinhallinnalla (apt-get, apt-cache)

Lähde: Karvinen, T. 2020. Command Line Basics Revisited. Luettavissa: https://terokarvinen.com/2020/command-line-basics-revisited/?fromSearch=command%20line%20basics%20revisited. Luettu: 22.1.2026.

b) Apt. Asenna kolme itsellesi uutta komentoriviohjelmaa
Valitsin kolme ohjelmaa(btop, ncdu ja tree). Asensin kaikki ohjelmat yhdellä komennolla: "sudo apt install -y htop ncdu tree". btop näyttää käynnissä olevat prosessit graaffisemmin. ncdu näyttää hakemistojen koon selkeämmin ja tree tulostaa nätimmin hakemistorakenteen.
